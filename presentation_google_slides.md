# Google スライド転用用原稿（16枚）

使い方:
1. Google スライドで「空白」のプレゼンテーションを新規作成
2. 各「スライド N」ブロックを **1枚のスライド** に対応させてコピー
3. デザイン: 背景 #f8fafc または白、見出し下に赤→オレンジ→黄の細線（任意）
4. デモスライドは **スクショ貼付** または **ハイパーリンク**（下記URL）

公開URL（リンク・スクショ元）:
- ダッシュボード: https://ibukiyamagiwa-dot.github.io/city-heat-dashboard/prototype_app.html
- 組織図: https://ibukiyamagiwa-dot.github.io/city-heat-dashboard/team_dashboard.html

## Google スライド作成のヒント

| 項目 | 推奨 |
|------|------|
| スライドサイズ | ワイド画面 16:9 |
| フォント | Yu Gothic / Noto Sans JP、本文 18〜24pt、タイトル 32〜40pt |
| 色 | 見出し #1f2937、補足 #64748b、アクセント #e60012 |
| デモ8 | スクショ + リンクボタン「ライブデモを開く」 |
| 発表時 | スライド8で別タブを開いておくとスムーズ |

## 枚数を減らす場合

省略候補: スライド11（駅別）→ 口頭のみ  
必須: 1, 2, 4, 7, 8, 16
